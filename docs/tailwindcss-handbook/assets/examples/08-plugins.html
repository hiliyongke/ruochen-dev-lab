多行截断（line-clamp-*）

新闻摘要

Tailwind CSS 提供 line-clamp-* 工具用于多行文本截断。在较新的版本中， 该功能已纳入核心工具，可以在无插件的前提下直接使用，例如 line-clamp-2、line-clamp-3。 结合响应式/交互变体，可以在不同断点展示不同的摘要行数。
