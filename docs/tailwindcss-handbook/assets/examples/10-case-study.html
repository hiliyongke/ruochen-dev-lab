FAQ

CDN 能用于生产吗？

不建议。生产请使用构建链路，启用内容裁剪与插件体系。

如何统一设计体系？

使用 theme.extend 与 CSS 变量结合，约束任意值，沉淀 presets。

类名太多怎么办？

@layer components 封装稳定组合，页面层保持原子类组合和清晰顺序。
